Barbearia Alura

Releitura do site desenvolvido durante os cursos de HTML5 e CSS3 da Alura. Através desse projeto, foi possível abordar várias ferramentas do HTML e do CSS, tais como: efeito parallax, efeitos de sombreamento ao clicar em itens como formulários e botões, troca de cores no texto clicado pelo mouse, dentre outros. Além disso, foi abordado o uso de tags semânticas no que se refere ao HTML.

Alura Books

Site desenvolvido durante o curso "HTML e CSS: responsividade com mobile-first" da Alura. As tecnologias utilizadas foram: HTML, CSS e JavaScript (para trabalhar com a biblioteca Swiper, responsável pelo carrossel de livros no banner do site). Como o próprio título do curso já entrega, foi usada a metodologia mobile-first; ela consiste em fazer o projeto primeiro para a versão mobile e, após isso, é que o projeto é adaptado para a versão desktop.
